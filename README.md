# ShowSphere

iOS app for browsing TV shows using the TVMaze API.

## Technology

- **Language**: Swift
- **Platform**: iOS, macOS, iPadOS (multiplatform app)
- **IDE**: Xcode
- **UI**: SwiftUI
- **Framework**: Observation framework
- **Testing**: SwiftTesting
- **Storage**: FileManager

## Project Structure

- `ShowSphere/` - Main app code
- `ShowSphereTests/` - Unit tests
- `ShowSphere.xcodeproj/` - Xcode project

## Build & Run

- Open `ShowSphere.xcodeproj` in Xcode
- Press ⌘R to run
- Press ⌘U to test

## Key Modules

- **CacheAndNetworkHelper** - Network requests and caching
- **Shows** - UI for show listing and details
